## Development Setup

```bash
# Prerequisites
node >= 18.0.0
npm >= 9.0.0

# Installation
git clone https://github.com/FartyLand/farty-platform
cd farty-platform
npm install

# Development
npm run dev

# Production Build
npm run build
```
